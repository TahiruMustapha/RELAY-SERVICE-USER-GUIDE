# Claude Code Comprehensive User Guide

This user guide provides comprehensive documentation for Claude Code, covering installation, best practices, configuration, custom models, and frequently asked questions.

## 🚀 Getting Started

If you are new to Claude Code, we recommend starting with the [Installation Guide](/docs/installation) to set up your environment on Windows, macOS, or Linux.

## ✨ Core Concepts

### 1. [Installation & Setup](/docs/installation)
Learn how to install Node.js, the Claude Code CLI, and configure your environment variables to connect to your proxy service.

### 2. [Best Practices](/docs/best-practices)
Master the art of writing effective prompts. Learn how to:
- Initialize projects and understand codebases.
- Develop new features and write tests.
- Debug and optimize your code efficiently.

### 3. [CLAUDE.md Configuration](/docs/claude-md)
Use a `CLAUDE.md` file to set persistent rules, roles, and workflows for Claude within your project. This ensures consistency and high-quality output.

### 4. [Custom Model Providers](/docs/custom-models)
Configure OpenClaw to connect to various AI model services like PinCC, Alibaba Cloud Bailian, DeepSeek, and more.

### 5. [Frequently Asked Questions](/docs/faq)
Find answers to common questions about platform support, language compatibility, troubleshooting, and security.

## 💡 Quick Tips for Efficiency

- **Use @ to reference files**: Direct Claude to specific files (e.g., `@src/main.ts`) to provide better context.
- **Extended Thinking**: Press `Shift+TAB` twice to enter **PLAN mode** for more thorough analysis.
- **Clean Context**: Use `/clear` and `/compact` to manage your conversation history and keep Claude focused.
- **Batch Operations**: Handle multiple similar tasks at once to save time and API calls.
